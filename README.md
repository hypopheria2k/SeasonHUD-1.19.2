# SeasonHUD (1.19.2)

A lightweight, highly customizable HUD for Minecraft 1.19.2 that enhances the **Serene Seasons** experience. It provides essential data for farming, survival, and immersion without cluttering the screen.

## ðŸš€ Features

* **Advanced Season Display**: Supports both standard seasons and tropical (Wet/Dry) biome logic.
* **Time Tracking**: Shows the current day, precise sub-season progress (%), and a countdown of remaining days until the next season.
* **Climate Monitoring**: Real-time display of biome temperature and humidity levels.
* **Fertility Indicator**: Dynamic icon (Poppy/Dead Bush) showing if the current season and biome allow for crop growth.
* **In-Game Editor**:
    * **Snap-to-Grid**: Smart 4-pixel screen edge snapping.
    * **Fully Scalable**: Adjust the HUD size to your preference.
    * **RGB Customization**: Define custom colors for every season and the info line via config.
* **Performance Optimized**: Designed for systems without dedicated GPUs. Minimal impact on FPS (Tested on AMD Ryzen 5600G Vega 7 / Linux Mint).
